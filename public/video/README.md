# Video File Needed

Please add your video file here named `placeholder.mp4`.

The video should be:
- Format: MP4 (H.264 codec recommended for best browser compatibility)
- Aspect Ratio: 16:9
- Recommended Resolution: 1920x1080px (1080p) or 1280x720px (720p)
- File Size: Optimized for web (recommend < 50MB)

The VideoSection component is already set up to display your video with native HTML5 controls.

If you want to add a poster image (thumbnail shown before video plays):
- Add an image file at `/public/images/video-poster.jpg`
- Dimensions: 1920x1080px
- Format: JPG or PNG
